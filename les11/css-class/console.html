Console Commands


  •   Console Openen
  •   Windows
    Typ 'cmd' in start
    of Windows Key + R 'cmd' en druk op ENTER
  •   Linux
    Open het menu en ga naar
    Applications ➝ Hulpmiddelen ➝ Terminal Venster
  •   macOS
    Druk op het Launchpad-symbool en typ 'Terminal'


  •   BestandsLocatie
  •   Windows
    'cd (BestandsLocatie)'
  •   Linux
    '$ cd (BestandsLocatie)'
  •   macOS
    'cd (BestandsLocatie)'


  •   Pushen Naar GitHub
  • Command 1 :
    echo "# test" >> README.md
  • Command 2 :
    git init
  • Command 3 :
    git add README.md
  • Command 4 :
    git commit -m "first commit"
  • Command 5 :
    git branch -M main
  • Command 6 :
    git remote add origin https://github.com/username/repo_name.git
  • Command 7 :
    git push -u origin main
