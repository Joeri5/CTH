  Map aanmaken via console

Uitleg hoe je mappen moet aanmaken
vanuit je console op elk OS.


Stap 1 : open Command Prompt door 'cmd' te typen in start of Windows Key + R 'cmd' en druk op ENTER
Stap 2 : ga naar de locatie waar je de map wilt hebben met het command:
'cd (BestandsLocatie)'
Stap 3 : maak een map aan met het command:
'mkdir (Map Naam)'

Stap 1 : open Terminal, open het menu en ga naar Applications ➝ Hulpmiddelen ➝ Terminal Venster
Stap 2 : ga naar de locatie waar je de map wilt hebben met het command:
'$ cd (BestandsLocatie)'
Stap 3 : maak een map aan met het command:
'mkdir (Map Naam)' of 'rmdir (Map Naam)'

Stap 1 : open de Terminal, druk op het Launchpad-symbool en typ 'Terminal'
Stap 2 : ga naar de locatie waar je de map wilt hebben met het command:
'cd (BestandsLocatie)'
Stap 3 : maak een map aan met het command: 'mkdir (Map Naam)'
